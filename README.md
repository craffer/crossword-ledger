# Crossword Ledger

A simple, static website to track and visualize the best (and worst) crossword puzzles that the NYTimes has published in the past few years, using data from a daily poll on [r/crossword](https://www.reddit.com/r/crossword).

Data is compiled and provided by [`7x11x13`](https://7x11x13.xyz/crossword/), AKA [u/Agitated-Garlic-2717](https://www.reddit.com/user/Agitated-Garlic-2717); see [his post here](https://www.reddit.com/r/crossword/comments/1eq057j/rcrossword_nyt_poll_archive/) for more details!

That's the brains of the operation; this is just a simple site on top to visualize the results.
